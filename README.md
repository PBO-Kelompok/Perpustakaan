Tugas UAS Kelompok 2 PBO dengan tema aplikasi pengelola buku pada perpustakaan 
untuk anggot akelompok kita terdiri dari 7 orang dengan nama nama berikut:
Maksima Insan Arafani / 25051204438 

Untuk bahasa yang digunakan pada projek kita ini adalah java swing
cara untuk running program kita :
buka terminal pada folder src lalu masukkan perintah
javac Main.java 
java Main
atau bisa juga dengan melakukan running menggunakan extension code runner
